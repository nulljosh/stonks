Portfolio Tracking

Real-time tracking of your holdings with automatic sector allocation analysis and performance metrics.

HFT-Style Streaming

Live price updates every second with volatility detection and momentum indicators for fast-moving opportunities.

Price Alerts

Set custom price targets and get instant notifications when your stocks hit key levels.

Stock Screener

Filter stocks by volume, P/E ratio, market cap, and distance from 52-week highs. Find opportunities that match your criteria.

Technical Indicators

RSI, MACD, and volatility metrics calculated in real-time to help identify entry and exit points.

Multi-Asset Support

Track stocks, crypto (BTC, ETH), and commodities (gold, silver) all in one dashboard.

Quick Start

# Clone the repository
git clone https://github.com/nulljosh/stonks.git
cd stonks

# Install dependencies
pip3 install -r requirements.txt

# Edit your portfolio (portfolio.json)
{
    "AAPL": {"shares": 0.3042},
    "BTC-USD": {"shares": 0.5},
    "GC=F": {"shares": 1.0}
}

# Run the tracker
python3 portfolio_tracker.py

# Run in streaming mode (real-time updates)
python3 portfolio_tracker.py --stream

Configuration

Customize alerts, watchlist, and screener criteria in config.json:

{
    "portfolio_url": "https://your-portfolio-site.com",
    "watchlist": ["AAPL", "NVDA", "BTC-USD"],
    "alerts": {
        "AAPL": {
            "above": 260,
            "below": 240
        }
    },
    "screener_criteria": {
        "min_volume": 1000000,
        "max_pe": 50,
        "min_market_cap": 10000000000
    }
}
