stonks

Terminal-based stock portfolio tracker inspired by mop

Python 3 yfinance Manifold Markets API colorama requests mop-style Terminal UI
View on GitHub Learn More

Market Indices

Live data for Dow, S&P 500, NASDAQ, Tokyo, HK, London, 10-Year Yield, Euro, Yen, Oil, and Gold at the top of every view.

Portfolio Tracking

Real-time holdings with color-coded gains (green) and losses (red) for instant visual feedback.

Mop-Style Display

Clean terminal table showing Last, Change, Change%, Open, Low, High, 52w Low, and 52w High for each position.

Multi-Asset Support

Track stocks, crypto (BTC-USD, ETH-USD), commodities (GC=F, SLV), and ETFs (IAU) all in one view.

Terminal Interface

Fast, lightweight terminal UI that runs anywhere Python runs. No browser, no bloat.

Simple Configuration

Just edit portfolio.json with your holdings and run. No complex setup or authentication required.

Prediction Markets

View trending prediction markets from Manifold Markets with live YES/NO probabilities, volume, and resolution dates.

Quick Start

# Clone the repository
git clone https://github.com/nulljosh/stonks.git
cd stonks

# Install dependencies
pip3 install -r requirements.txt

# Edit your portfolio (portfolio.json)
{
    "AAPL": {"shares": 0.3042},
    "HOOD": {"shares": 0.0229},
    "SLV": {"shares": 0.5251}
}

# Run the tracker
python3 portfolio_tracker.py

Configuration

Customize alerts, watchlist, and screener criteria in config.json:

{
    "portfolio_url": "https://your-portfolio-site.com",
    "watchlist": ["AAPL", "NVDA", "BTC-USD"],
    "alerts": {
        "AAPL": {
            "above": 260,
            "below": 240
        }
    },
    "screener_criteria": {
        "min_volume": 1000000,
        "max_pe": 50,
        "min_market_cap": 10000000000
    }
}
